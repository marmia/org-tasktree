#+TITLE: org-tasktree Specification Index
#+STARTUP: showall

* 目的
本ドキュメントは org-tasktree の確定済み仕様のみを整理した仕様書である。
バージョン別の作業項目や未確定事項は plan 配下に記載する。

* DB 変更時の運用
- スキーマ変更がある場合は、開発環境の DB を手動で再初期化する。
- 移行スクリプトは作成しない前提とする。
