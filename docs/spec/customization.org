#+TITLE: org-tasktree Specification - Customization
#+STARTUP: showall

* カスタマイズ変数一覧

** org-tasktree-database-location

- 型: string
- 説明: SQLite データベースファイルのパス
- 初期値の決定方法:
  - 初回起動時に以下のパスが自動設定される。

    (expand-file-name "org-tasktree/tasktree.db" user-emacs-directory)

- 例:
  - ~/.emacs.d/org-tasktree/tasktree.db
  - ~/.config/emacs/org-tasktree/tasktree.db

- ユーザーが後から任意の場所に変更してもよい。

** org-tasktree-query-dir

- 型: string
- 説明: クエリー YAML ファイルの保存先ディレクトリ
- 初期値の決定方法:
  - 初回起動時に以下のパスが自動設定される。

    (expand-file-name "org-tasktree/queries/" user-emacs-directory)

- 例:
  - ~/.emacs.d/org-tasktree/queries/
  - ~/.config/emacs/org-tasktree/queries/

- ディレクトリは org-tasktree-init 実行時に自動作成される。

** org-tasktree-ui-minibuffer-completion-color-domain

- 型: color
- 説明: `org-tasktree-find-node` の domain candidate 表示色
- 初期値: #40C97E

** org-tasktree-ui-minibuffer-completion-color-project

- 型: color
- 説明: `org-tasktree-find-node` の project candidate 表示色
- 初期値: #66BFFF

** org-tasktree-ui-minibuffer-completion-color-phase

- 型: color
- 説明: `org-tasktree-find-node` の phase candidate 表示色
- 初期値: #F08AC2

** org-tasktree-ui-minibuffer-completion-color-group

- 型: color
- 説明: `org-tasktree-find-node` の group candidate 表示色
- 初期値: #FFB366

** org-tasktree-ui-minibuffer-completion-color-task

- 型: color
- 説明: `org-tasktree-find-node` の task candidate 表示色
- 初期値: white
