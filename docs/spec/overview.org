#+TITLE: org-tasktree Specification - Overview
#+STARTUP: showall

* 概要
org-tasktree は、org-mode を編集 UI とし、SQLite を唯一のマスターデータベースとする
Emacs 用タスク管理プラグインである。

- org バッファは常に「編集用ビュー」
- SQLite データベースが single source of truth
- DB 更新は UPDATE / INSERT を使用し、既存レコードの id を維持する

* 基本コンセプト

- ノードの同一性は UID のみで保証する
- parent_id は org バッファには保持しない

* タスクの階層構造
- ノードの階層構造に制約は設けない
- 任意のノード配下に任意のノードを作成できる
